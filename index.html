Quién eres y tu experiencia

Iniciamos como comunidad de código abierto de GitHub para el desarrollo de sitios y aplicaciones web de código libre.

¿Quién es tu público?

  • Padres (solteros, casados o unión libre) de uno a tres hijos.
  • Empresas (el SENA).
  • Publico profesional.

¿Qué problema tiene?

muchas empresas de desarrollo web no sigue los estándares actuales de la web.
en algunos casos muchos estudios fotográficos no le cumplen con las experiencias con los clientes.

¿Qué solución ofreces?

Ofrecemos una gran variedad de soluciones para generar experiencias de acuerdo con las especificaciones de cliente.
